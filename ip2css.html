CSS INTERVIEW QUESTIONS DAY 2

1.What is box sizing?

The CSS box-sizing property allows us to include the padding and border in an element's total width and height.
When you set the width or height of an element, the element often appears bigger than you have set (because the elements border and padding are added to the elements specified width or height).

2.What is animation delay?

The animation-delay property specifies a delay for the start of an animation.
Negative values are also allowed. If using negative values, the animation will start as if it had already been playing for N seconds.

4.What is grid-template-columns used for?

The grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column. The value is a space-separated-list, where each value defines the width of the respective column.

7.what is the difference between display none and visibility hidden?

display:none is commonly used to hide and show elements without deleting and recreating them. visibility:hidden also hides an element. However, the element will still take up the same space as before. The element will be hidden, but still affect the layout.
